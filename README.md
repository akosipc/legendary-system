## Shopify Bridge

#### Version

```
  React 18.2.0
  Nextjs 12.2.2
  Typescript 4
  ChakraUI 2.2.4

  Nodejs 16.4.0
  Yarn 1.22.17
```

#### Installation

```
  $ git clone git@github.com:akosipc/legendary-system.git
  $ cd legendary-system
```

#### Setup

```
  $ yarn install
```

#### Starting the application

```
  $ yarn run dev
```

#### How to run the test suite

```
  $ yarn run tsc
```
